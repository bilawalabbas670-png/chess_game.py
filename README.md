# 🎮 Game Project (Python)

This project contains two simple Python games developed for learning programming concepts.

---

## ♟️ 1. Chess Game
A basic console-based chess board implementation.

### Features:
- 8x8 chess board display
- Basic piece setup
- Move system (simple input-based)
- Turn-based gameplay

---

## ❌⭕ 2. Tic Tac Toe Game
A simple 2-player game built in Python.

### Features:
- 3x3 game board
- Player X and O turns
- Win detection (rows, columns, diagonals)
- Draw condition

---

## 🧠 Technologies Used:
- Python
